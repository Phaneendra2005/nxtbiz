# NxtBiz

NxtBiz is a spec-driven business operations automation platform with CRM, email intelligence, workflow execution, AI-style agent orchestration, realtime notifications, and PDF generation.

The codebase follows the spec-defined structure:

- `client/` for the Vite React operations console
- `server/` for the modular Express API

## Phase 1 Status

This initial implementation establishes the application foundation: package metadata, backend API shell, authentication, role middleware, database models, seed data, and the frontend app shell with protected routes.
